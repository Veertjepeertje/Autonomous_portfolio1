# Autonomous Systems Portfolio 1 – Rule-Based Agents

## Over het project
In dit project zijn meerdere rule-based agents ontwikkeld voor het spel Connect Four met behulp van de PettingZoo omgeving.

Het doel is om verschillende strategieën te ontwerpen en te vergelijken op basis van hun prestaties.

## Agents
De volgende agents zijn geïmplementeerd:

- Baseline Agent (random)
- MiddleFirst Agent
- HorizontalPreference Agent
- Blocking Agent
- ForkBuilder Agent

Elke agent maakt gebruik van een eigen set regels om beslissingen te nemen.

## Installatie
Clone de repository:

git clone https://github.com/Veertjepeertje/Autonomous_portfolio1.git

Ga naar de map:

Autonomous_portfolio1

Installeer de benodigde packages:

pip install -r requirements.txt

## Gebruik
Open het notebook:

portfolio1.ipynb

Run alle cellen om:
- agents tegen elkaar te laten spelen
- resultaten te genereren
- visualisaties te zien

## Resultaten
De prestaties van de agents worden gemeten met:
- aantal wins
- aantal draws
- winrate

Deze worden weergegeven met heatmaps en grafieken.
Deze visualisaties maken het mogelijk om de prestaties van de agents eenvoudig te vergelijken en te analyseren.

## Evaluatie
Elke agent speelt meerdere games tegen alle andere agents.  
Om eerlijke resultaten te krijgen wisselen de agents van startpositie.
